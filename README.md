# Serenité — Beauty & Spa Website

A sleek, modern single-page website for a luxury beauty & spa salon. Built with React, TypeScript, and Tailwind CSS, featuring smooth GSAP scroll-triggered animations throughout.

## ✨ Features
- **Hero** with parallax background and staggered text reveals
- **About** section with slide-in animations
- **Services & Pricing** grid with staggered card entrances
- **Gallery** with clip-path curtain-wipe image reveals
- **Booking/Contact** form with sequenced animations
- Fully responsive design (mobile, tablet, desktop)
- Fixed navbar with scroll-aware backdrop blur

## 🛠 Tech Stack
- React + javaScript + Vite
- Tailwind CSS
- GSAP + ScrollTrigger for animations
